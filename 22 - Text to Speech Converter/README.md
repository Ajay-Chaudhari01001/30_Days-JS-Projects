# Text To Speech Convertor
This is a simple text to speech convertor, We can convert our text into various sounds like US Female and Male Sound and outhers country fluency

# Fetures:
1) Convert Text To Speech,
2) PAUSE The Sound,
3) RESUME The Sound,
